# Roadmap

Planned future improvements for IdeaCommand:

- **Improve navigation** — Smoother movement between ideas with parent/up navigation, breadcrumb trails, quick sibling jumps, and better tree context display
- **Redesign the workflow** — Consolidate commands into a more unified experience with fewer discrete steps
- **Enhanced questions** — More sophisticated question flow for defining ideas with better context and guidance
- **Naming convention control** — Rules and validation for idea naming patterns
- **Improved searching** — Faster, more flexible search across the tree with filtering and content search
- **Combine find with new** — Search before creating to prevent duplicates
- **REPL help command** — Built-in help inside the REPL
- **REPL tab completion** — Tab auto-complete for commands and idea names
- **Insert parent command** — Insert a parent node between existing parent and child without unlinking
